### c-button 用法

> 用法
```
<c-button size="x-small" type="primary" icon="back" disabled @click="method">value</c-button>
```

> 参数说明

- class 样式
- disabled 禁用
- icon 图标
- size 默认normal
    - x-large 
    - large
    - normal
    - small
    - x-small
- type 默认default
    - primary
    - warning
    - default
－ value 名称

> other说明

- 可以直接写其它button属性、事件`@click`等
